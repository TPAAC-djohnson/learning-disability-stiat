Single-Target Implicit Association Test (ST-IAT)
Stimulus Specification – Learning Disability (Word-Only)

PURPOSE
This ST-IAT assesses automatic affective evaluations toward the single target concept
“Learning Disability” using word-based stimuli and evaluative attributes (Pleasant vs.
Unpleasant). The task is adapted from a previously approved dual-target IAT and revised
to a single-target format to improve construct clarity and interpretability.

TARGET CATEGORY (SINGLE TARGET)
Learning Disability

TARGET STIMULI (WORDS)
Learning disability
Intellectual disability
Developmental delay
Cognitive impairment
Special education

ATTRIBUTE CATEGORY: PLEASANT
Joy
Warmth
Hope
Comfort
Love
Peace

ATTRIBUTE CATEGORY: UNPLEASANT
Pain
Fear
Burden
Stress
Suffering
Difficulty

PARTICIPANT INSTRUCTIONS
In this task, you will be asked to categorize words as quickly and accurately as possible.

Use the E key for items that belong to the category shown on the left side of the screen.
Use the I key for items that belong to the category shown on the right side of the screen.

If you make an error, an X will appear. Please correct the error and continue.

This task requires a desktop or laptop keyboard.

ST-IAT BLOCK STRUCTURE

Block 1 – Attribute Practice
Left: Pleasant
Right: Unpleasant
Stimuli: Attribute words only

Block 2 – Combined Practice
Left: Pleasant + Learning Disability
Right: Unpleasant
Stimuli: Attribute words + target words

Block 3 – Combined Test
Left: Pleasant + Learning Disability
Right: Unpleasant
Stimuli: Attribute words + target words

Block 4 – Reversed Practice
Left: Pleasant
Right: Unpleasant + Learning Disability
Stimuli: Attribute words + target words

Block 5 – Reversed Test
Left: Pleasant
Right: Unpleasant + Learning Disability
Stimuli: Attribute words + target words

RESPONSE MAPPING
Left key: E
Right key: I

IMPLEMENTATION NOTES
Word-only stimuli
Reaction times and errors recorded automatically
Implemented in Qualtrics XM using JavaScript-enabled Text Entry questions
No personally identifiable information collected
